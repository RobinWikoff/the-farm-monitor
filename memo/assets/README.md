# The Farm Memo Logo Asset

Save the standard memo logo at:

memo/assets/the_farm_logo.png

## Recommended design target

- Aspect ratio: 4:1 (width:height)
- Layout: [image mark 2:1] + [text mark "The Farm" 2:1]
- File format: PNG (preferred), transparent background

## Recommended export sizes

- Primary: 1600 x 400 px (4:1)
- High-res print option: 2400 x 600 px (4:1)

## Resolution guidance

- For print-quality source artwork: design/export at 300 PPI equivalent
- For on-screen use in this template, pixel dimensions above are sufficient

The memo generator will use this file automatically if present.
